# 🕵️‍♀️ NetScanner: Network Security Tool

Welcome to **NetScanner**! This is a simple, real-time web application built with Python and Streamlit to scan IP addresses for open vulnerable ports.

I built this project to practice my Networking (Réseaux) and IT Security skills, bridging the gap between Python development and Cybersecurity.

## ✨ Features
* **Real-time Port Scanning:** Checks for common vulnerable ports (e.g., FTP, SSH, HTTP, HTTPS).
* **Matrix-Style UI:** A dark-themed, hacker-style interface built with Streamlit.
* **Instant Feedback:** Tells you immediately which ports are OPEN (vulnerable) and which are CLOSED (secure).

## 🛠️ Tech Stack
* **Language:** Python
* **Networking:** `socket` library
* **Frontend/UI:** Streamlit
